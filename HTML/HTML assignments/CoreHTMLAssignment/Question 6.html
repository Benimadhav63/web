1. HTML
    Primary use case Key Features
    Building the structure of web pages
    • simple and easy to learn
    • Compatible with all web browser
    • Allows for semantic Markup
    Internet navigation
    • uses hypertext concept for navigation
    • user is directed to the referenced text or website
    • A user can move between web pages and websites
  2. CSS
    Primary use case Key Features
    Website Styling
    • CSS is primarily used to style websites
    • allowing developers to create visually appealing
    • consistent user interfaces across different web pages
    Responsive Web Design
    • CSS is used to make web pages responsive
    • adapting the layout and appearance based on the device' screen size
  3. JavaScript
    Primary use case Key Features
    Validating User's Input
    • JavaScript is very useful while using form
    • It has the capability to validate user input for errors and also saves time
    Simple Client-side Calculations
    • it can perform basic calculations on the browser
    • The browser does not need to ask server time for every task
  4. ReactJs
    Primary use case Key Features
    JavaScript Syntax Extension
    • JSX is a combination of HTML and JavaScript
    • You can embed JavaScript objects inside the HTML elements
    Simplicity
    • React.js is a component-based which makes the code reusable
    • This makes code easy to understand and easy to debug and has less code
  5. Express
    Primary use case Key Features
    Single-page applications(SPAs)
    • Express.js can be used to build SPAs
    • provide a seamless user experience by loading all the necessary resources on a single page.
    Real-time applications
    • Express.js can be used to build real-time applications
    • require constant communication between the client and server
  6. NodeJs
    Primary use case Key Features
    Single-Threaded
    • Node.js employs a single-threaded architecture with event looping
    • the event mechanism allows the node.js server to reply in a non-blocking manner and makes it more scalable
    Scalable
    • NodeJs can handle concurrent requests efficiently
    • It has a cluster module that manages load balancing for all CPU cores that are active
  7. MongoDB
    Primary use case Key Features
    Authentication
    • Authentication is a critical security feature in MongoDB
    • Authentication ensures that only authorized users can access the database
    Database Triggers
    • Database triggers in MongoDB Atlas are a powerful feature that allow you to execute code when certain events occur in your database
    • MongoDB Atlas allows you to create and manage triggers in a simple, intuitive way
